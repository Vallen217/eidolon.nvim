## Installation

Install the theme via package manager, such as [packer.nvim](https://github.com/wbthomason/packer.nvim)

```lua
use("Vallen217/eidolon.nvim")
```

## Usage

```lua
vim.cmd[[colorscheme eidolon]]
```
